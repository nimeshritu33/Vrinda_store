# Vrinda_store

Sales Data Analysis Project (2022)
📌 Project Overview
This project focuses on analyzing sales data for the year 2022 to extract meaningful insights. The goal is to understand customer behavior, sales trends, and business performance using data visualization and analysis techniques.
🎯 Objectives
The main objectives of this project are:
Compare sales and orders using a single chart
Identify the month with highest sales and orders
Analyze purchase behavior (men vs women)
Explore different order statuses in 2022
List top 10 states contributing to sales
Understand relationship between age and gender (number of orders)
Identify top-performing sales channels
Find the highest selling category
📂 Dataset
Year: 2022
Includes:
Sales data
Order details
Customer demographics (age, gender)
State-wise data
Sales channels
Product categories
📊 Tools & Technologies Used
Microsoft Excel
Data Cleaning & Preprocessing
Pivot Tables
Charts & Data Visualization
📈 Key Analysis Performed
1. Sales vs Orders Comparison
Combined chart (Bar + Line) to visualize trends
2. Monthly Performance
Identified peak month for sales and orders
3. Gender-based Analysis
Compared purchasing behavior of men vs women
4. Order Status Breakdown
Categories like:
Delivered
Cancelled
Returned
5. Top 10 States
Ranked states contributing highest to total sales
6. Age vs Gender Analysis
Distribution of orders based on age groups and gender
7. Sales Channel Contribution
Platforms contributing maximum revenue
8. Category Performance
Identified best-selling product category
